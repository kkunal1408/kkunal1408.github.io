# Kishor Kunal — personal website

Source for [kkunal1408.github.io](https://kkunal1408.github.io), a lightweight personal bio and research portfolio built with semantic HTML, CSS, and a small amount of JavaScript.

## Local preview

Open `index.html` directly, or run a local static server:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
